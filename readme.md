# Зависимости #
1. yarn
2. react
3. npm
4. vite

# Установка #
Клонировать ветку base.
```
git clone --branch=base https://github.com/veoutlo/trapezoid.git
```

Установить все зависимости проекта:
```
yarn install
```

Запуск проекта
```
yarn dev
```
